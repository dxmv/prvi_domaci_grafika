
### Stars
- Twinkle
- Mulitple colors
- Move around 
- Bounce

### Planets
- 2 different spritesheets 
- each sprite has 50 frames
- goes in a circle

### Ship
- Simple triangle
- Reacts to controls, rotations on A/D, move forward
- Color change when hit by enemy

### Lasers
- Handles user input
- Shoots from direction of player
- Expands until some width
- Gets thicker
- To add cool gradient effect

### Enemies
- Basic red squares
- Spawn randomly
- Follow the player
- Have an alpha and scale animation, alpha and scale gets smaller when they're dead

### Animations
- Collison laser - enemy
- Collison player - enemy
